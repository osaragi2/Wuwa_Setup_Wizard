# Wuthering Waves Setup Wizard - Release Notes (v1.0.1)

## 🐛 Bug Fixes
- **Sửa lỗi Crash Shader:** Khắc phục tình trạng treo máy hoặc lỗi hiển thị khi áp dụng Shader trong trường hợp thiếu file dữ liệu `.blend`.
- **Sửa lỗi Eye Tracker:** Khắc phục lỗi khiến tính năng theo dõi mắt (Eye Tracker) bị mất kết nối và không hoạt động sau khi sử dụng công cụ Import Face Panel.

## ✨ New Features
- **Nâng cấp Global Settings (Cài đặt Toàn cục):** Bổ sung thêm nhiều thông số tùy chỉnh mới cho Shader, cho phép thay đổi trực tiếp từ bảng điều khiển:
  - **Tùy chỉnh Bóng đổ (Shadow):** Hỗ trợ tinh chỉnh Shadow Position (Vị trí bóng), Shadow Softness (Độ mềm của bóng) và Enable Cast Shadows (Bật/tắt bóng đổ).
  - **Tùy chỉnh Màu da (Skin Color):** Hỗ trợ thay đổi Base Color (Màu nền), Shadow Color (Màu vùng tối) và Skin Color Multiplier (Cường độ màu da).
